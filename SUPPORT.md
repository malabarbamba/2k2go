# Support

## Get Help

- Usage questions: open a GitHub issue with clear context and reproduction steps.
- Bug reports: use the bug issue template.
- Feature ideas: use the feature request template.

## Security Issues

Do not file public issues for vulnerabilities.

Follow `SECURITY.md` to report privately.

## What to Include

For faster help, include:

- Expected behavior.
- Actual behavior.
- Reproduction steps.
- Relevant environment details.
- Logs/screenshots with secrets and personal data removed.
